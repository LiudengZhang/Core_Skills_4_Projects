Core Skills for Claude Code Projects 4 skills

Reusable Claude Code skills for project management, task orchestration, and session tracking.

▦
/dashboard
Generate an HTML dashboard showing current task status, blockers, agent components, recent conversations, and project notes.
  • Multi-agent component tracking (each agent writes its own JSON)
  • Task status synced with TaskList
  • Parses conversation history from local JSONL files
  • Integrates project notes grouped by plan
  • Auto-refresh single-page HTML output
⚙
/dispatch
Create and manage task plans for multi-agent execution. Commander/Executor pattern where one terminal creates plans and others execute them.
  • /dispatch create — structured interview to build a plan
  • /dispatch execute — executor picks up and runs a plan
  • Completion checklists (figures, analysis, writing)
  • Context files for shared project knowledge
  • Dashboard integration on plan completion
⏱
/history
Browse and search conversation history from JSONL files. List sessions, summarize conversations, search for keywords across sessions.
  • /history list [n] — recent sessions with previews
  • /history summarize <id> — date range, topics, files modified
  • /history search "keyword" — cross-session search with context
  • Python helper with parse, summary, and search functions
✎
/notes
Manage project notes linked to tasks, plans, and components. Notes stored as JSON files and displayed on the dashboard grouped by plan.
  • /note add — link to tasks, plans, figures, components
  • 4 note types: finding, decision, blocker, question
  • /note show — filter by plan, task, type, or component
  • /note archive — move resolved notes out of active view
  • Python helper with full CRUD and grouped-by-plan queries

Installation

git clone https://github.com/LiudengZhang/Core_Skills_4_Projects.git
cp -r Core_Skills_4_Projects/core_dashboard  your-project/.claude/skills/dashboard
cp -r Core_Skills_4_Projects/core_dispatch   your-project/.claude/skills/dispatch
cp -r Core_Skills_4_Projects/core_history    your-project/.claude/skills/history
cp -r Core_Skills_4_Projects/core_notes      your-project/.claude/skills/notes

Each skill folder contains a SKILL.md file (and optional helpers) that Claude Code loads automatically from .claude/skills/.
